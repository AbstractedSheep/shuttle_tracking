To make a user an administrator:
- cd into the project directory
- Run the rails console using "rails c"
- In the rails console:

- Create a user using "user = User.create()"
- Set the username using 'user.username = "yourRCSIDhere"'
- Set administrator status using "user.is_admin = true"
- Save the user using 'user.save'
- Exit the rails console ('exit')
- Restart your server if it is running
